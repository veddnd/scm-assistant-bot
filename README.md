SCM Assistant – Supplier Network RAG Chatbot
Project Overview
SCM Assistant is a Retrieval-Augmented Generation (RAG) chatbot built using Flowise AI. The chatbot is designed to answer questions about a supplier network using supplier performance data and supply chain governance documents.
The solution uses document retrieval and Google Gemini models to provide context-aware responses based on the uploaded knowledge base.
